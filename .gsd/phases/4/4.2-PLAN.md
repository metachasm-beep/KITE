# Plan 4.2: User Account & Profile System

## Objective
Build the authenticated user-facing experience: a profile page displaying account information and order history, pulling live data from Prisma/Postgres.

## Context
- NextAuth provides session data (email, name, image, role).
- Prisma schema provides the Order history, Cart, and Address data.
