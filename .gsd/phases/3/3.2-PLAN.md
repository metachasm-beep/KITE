# Plan 3.2: Grid-Based Collections Interface

## Objective
Establish the primary public-facing collections view `(/collections)` that dynamically fetches content from the CMS Bridge (`lib/cms.ts`) and presents a styled grid of available artifacts.

## Context
- Connects to `lib/cms.ts` -> `getArtifacts()`
- Must adhere strictly to the VOIDLAB UI language: black, white, and `.accent`.
