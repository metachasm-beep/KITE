# Plan 2.1: Authentication Core

## Objective
Establish the primary authentication layer using NextAuth/Auth.js configured with Google Sign-in to distinguish between general users and administrators.

## Context
- .gsd/SPEC.md
- .gsd/ARCHITECTURE.md
- Goal: Secure login for platform usage.

## Tasks

<task type="auto">
  <name>Install Authentication Libraries</name>
  <files>package.json</files>
  <action>
    1. Install `next-auth` (latest version for Next.js App Router).
  </action>
  <verify>Check package.json for next-auth dependency.</verify>
  <done>next-auth is installed successfully.</done>
</task>

<task type="auto">
  <name>Implement NextAuth Route Handler</name>
  <files>app/api/auth/[...nextauth]/route.ts, lib/auth.ts</files>
  <action>
    1. Create the Auth Configuration file (`lib/auth.ts`).
       - Setup `GoogleProvider` using environment variables (`GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`).
       - Define a callback to hardcode an environmental `ADMIN_EMAIL` check. If the signed-in user matches `ADMIN_EMAIL`, append `role: "admin"` to the session object; otherwise, append `role: "user"`.
    2. Create the App Router API handler utilizing the config from `lib/auth.ts`.
  </action>
  <verify>Route handler exists and builds without errors.</verify>
  <done>App NextAuth routes are functional and ready for OAuth flow.</done>
</task>

<task type="auto">
  <name>Setup Session Provider</name>
  <files>app/layout.tsx, components/Providers.tsx</files>
  <action>
    1. Create a `Providers.tsx` component wrapping `SessionProvider`.
    2. Import and wrap the root layout children with the `<Providers>` component to grant session access across the Next.js app.
  </action>
  <verify>App builds and runs locally without context compilation errors.</verify>
  <done>Session context is available globally.</done>
</task>

## Success Criteria
- [ ] `next-auth` installed and configured.
- [ ] Google OAuth provider implemented with role validation capability.
- [ ] Next.js app successfully wrapped in Session context.
