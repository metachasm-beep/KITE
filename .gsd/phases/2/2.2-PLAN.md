# Plan 2.2: Admin Dashboard Layout & Protection

## Objective
Establish a protected `/admin` route utilizing Next.js layout composition to host the secure Admin Dashboard and verify user roles.

## Context
- .gsd/ARCHITECTURE.md
- Admin must be authenticated and their session config role must be "admin".
