# Plan 1.1: Project Setup & UI Foundation

## Objective
Initialize the Next.js project with Tailwind CSS and configure the brand's visual identity (Cyber Purple accent and Space Grotesk typography).

## Context
- .gsd/SPEC.md
- .gsd/ARCHITECTURE.md
- Brand Colors: #000000 (base), #8B5CF6 (Cyber Purple accent)
- Fonts: Space Grotesk (Headings), Inter (Body)

## Tasks

<task type="auto">
  <name>Initialize Next.js & Tailwind</name>
  <files>package.json, tailwind.config.ts, app/globals.css</files>
  <action>
    1. Initialize a new Next.js app in the root directory (if not exists).
    2. Configure `tailwind.config.ts` to include:
       - Theme colors: `accent: "#8B5CF6"` (Cyber Purple), `base: "#000000"`.
       - Font families: `sans: ["Inter", "sans-serif"]`, `heading: ["Space Grotesk", "sans-serif"]`.
    3. Update `globals.css` with a deep black background (#000000) and white text as default.
  </action>
  <verify>npm run dev (build check) and check tailwind.config.ts colors</verify>
  <done>Next.js initialized and Tailwind configured with Cyber Purple accent color.</done>
</task>

<task type="auto">
  <name>Configure Global Typography</name>
  <files>app/layout.tsx</files>
  <action>
    1. Import `Space_Grotesk` and `Inter` from `next/font/google`.
    2. Define fonts with proper subsets and variable names.
    3. Apply fonts to the root layout body.
  </action>
  <verify>Check browser for correct fonts in the rendered HTML.</verify>
  <done>Space Grotesk and Inter fonts are integrated and active in the layout.</done>
</task>

## Success Criteria
- [ ] Next.js app running.

- [ ] Space Grotesk typography applied to headings.
